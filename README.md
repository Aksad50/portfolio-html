# 🌐 Portfolio HTML

A clean and simple **personal portfolio website** built with pure HTML — no frameworks, no dependencies, just the web fundamentals.

---

## 📋 About

This is a personal portfolio project that showcases information about me — my background, skills, and biography. Built entirely with HTML, making it lightweight and easy to deploy anywhere.

## 🚀 Getting Started

### Prerequisites

All you need is a web browser — no installations required!

### Running Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aksad50/portfolio-html.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd portfolio-html
   ```

3. **Open in browser**
   ```bash
   # Simply open the HTML file
   open Biogrphy.html
   ```
   Or just double-click the `Biogrphy.html` file in your file explorer.
